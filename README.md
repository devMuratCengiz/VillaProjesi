# Proje Detayı
Bu proje Asp.Net kullanılarak geliştirilmiş bir Villa satış sitesidir. Sayfadaki her öğe dinamik haldedir.
## Kullanılan Teknolojiler
- ASP.NET
- MongoDB
- Entity Framework Code First
- MVC
- View Component
- Fluent Validation
